# My Blog

Personal blog built with Next.js. Write posts in markdown, they show up on the site.

## Setup

Install dependencies:
```bash
npm install
```

Run the dev server:
```bash
npm run dev
```

Open http://localhost:3000

## Adding Posts

Create markdown files in `content/posts/` with frontmatter:

```markdown
---
title: Your Post Title
date: 2024-01-15
tags:
  - Tag1
  - Tag2
excerpt: Short description
author: Your Name
---

Your content here...
```

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Markdown for posts

## Build

```bash
npm run build
npm start
```
